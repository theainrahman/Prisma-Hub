ABOUT US

Discovering Information Management
Resources in PSB Library

At Prisma Hub, we bridge the gap between complex research and academic excellence by disseminating vital information resources to higher education institutions.

01 OUR COMPANY

Prisma Hub Book Collection serves as a premier professional repository dedicated to disseminating authoritative academic literature and essential reference texts across Malaysia.

Each book is carefully selected to support learning, teaching, and research in the field of Information Management.

02 OBJECTIVES

OBJECTIVE 1

To provide information about Information Management books.

OBJECTIVE 2

To promote and increase awareness of library resources.

OBJECTIVE 3

To support student learning, research, and academic success.

OBJECTIVE 4

To apply web development skills using XHTML and CSS.

03 PROJECT FEATURES

BOOK INFORMATION

Detailed information for each selected book including title, author, publisher, and subjects.

BOOK GALLERY

Multiple images of each book including front cover, back cover, and contents preview.

EASY NAVIGATION

User-friendly design and clear layout for easy browsing and better experience.

EXTERNAL LINKS

Links to related resources and references for further reading and research.

04 DEVELOPMENT PROCESS

PLANNING

Define the project scope, objectives, and content structure.

ANALYSIS

Gather and analyze information about the selected books.

DESIGN

Plan the layout, navigation, and user interface using wireframes.

DEVELOPMENT

Build the website using XHTML for structure and CSS for styling.

TESTING

Test all pages, links, and functionality to ensure quality.

PUBLISHING

Deploy the website to GitHub and make it accessible online.

ABOUT THIS PROJECT

This website presents ten Information Management books available in Perpustakaan Sultan Badlishah. This project is for academic purposes only.
